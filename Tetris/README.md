# Tetris

**Controls**

Right arrow -> Move the piece to the right

Left arrow -> Move the piece to the left

Down arrow -> Drop the piece

Up arrow -> Rotate the piece clockwise

Z -> Rotate the piece counterclockwise

R -> Restart

P -> Pause/Unpause
